#  Sales Dashboard Analysis using Excel

##  Project Overview

This project presents a comprehensive sales analysis dashboard built using Microsoft Excel. The dashboard provides insights into sales performance across regions, product categories, and time trends to support data-driven decision making.

---

##  Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning & Transformation
* Slicers (for interactivity)
* KPI Metrics

---

##  Dataset Description

The dataset includes:

* Sales transactions
* Order details
* Product categories
* Regional data
* Monthly sales data

---

##  Dashboard Features

### 🔹 Key Performance Indicators (KPIs)

* Total Sales: ₹51,38,670
* Total Orders: 120
* Average Order Value: ₹42,822
* Total Discount: ₹4,30,130

---

###  Sales by Region

* Compares revenue across East, North, South, and West regions
* Helps identify high-performing regions

---

###  Sales by Category

* Analyzes performance of product categories:

  * Electronics
  * Home Appliances
  * Cooling Appliances

---

###  Sales Trend Analysis

* Monthly sales trend visualization
* Identifies growth patterns and seasonal changes

---

###  Interactive Filters (Slicers)

* Region
* Category
* Month

---

##  Key Insights

* South region contributes the highest revenue
* Electronics category generates approximately 60% of total sales
* Sales peaked in March and slightly declined in April
